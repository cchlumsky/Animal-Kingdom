# Animal-Kingdom

### Description
Match pets that need homes with people best suited to take care of them!

### Key Features
1) Human survey
2) Animal list
3) Matching algorithm
4) Swipable pics

### Tech Used
1) mysql
2) bulma
3) swipper
4) chia
5) mocha
6) express
7) sequalize
